Option allows checking and running missed patches when connection is established. If a new patchset has been created during Jenkins server down time, will Jenkins trigger patchset created event normally when connection is created.

Caution!
  • Allowing the property takes place immediately.
  • Disabling the property takes place after restarting Jenkins server.
  • Property supports only plain text formatted project patterns. Other type of project patterns will be ignored.
  • Property supports only patchset created events. Other type of events will be ignored.
